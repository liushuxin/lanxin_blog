## React Router 哲学

&emsp;&emsp;在构建单页面应用时，我们需要用到路由，来决定不同的 URL 跳转到不同的页面，而 React 的思想是以组件化的形式来构建页面，所以 React Router 是根据不同的 URL 跳转到不同的组件。

&emsp;&emsp;对于绝大部分框架来说，路由都是静态的，react-router 在 pre-v4 之前也是静态的，但是之后改为动态路由
动态路由是应用程序渲染时发生的路由
