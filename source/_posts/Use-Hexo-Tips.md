### 2，基本用法命令

```
$ hexo clean #清理之前生成的内容，即public文件
$ hexo g #生成静态文件
$ hexo d #部署
$ hexo s #启动本地服务，可以通过http://localhost:4000查看
$ hexo s --debug #使用 debug 模式启动服务
```

创建新文章方式：

```
$ hexo new [layout] <title>
```
